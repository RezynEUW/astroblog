## Usage in MDX

```jsx
import ImageGallery from '../../components/ImageGallery.astro';

// Grid layout
<ImageGallery 
  images={[/* your images */]}
  columns={2}  // 1, 2, or 3
/>

// Carousel layout
<ImageGallery 
  images={[/* your images */]}
  layout="carousel"
/>
```
